Project:   https://github.com/qew21/Genshin-Subtitles
License:   Apache License, Version 2.0
Notice:    Kaption began as a fork of qew21/Genshin-Subtitles. The codebase
           has since been substantially rewritten: the OCR pipeline, the
           three-stage matcher (SymSpell + n-gram + OCR-weighted
           Levenshtein), screen capture, layout engine, networking,
           licensing, security stack, and UI are all new work. The
           original repository is nonetheless the starting point this
           project grew out of, and that lineage is recorded here.

           This entry is listed alongside the bundled components for
           bookkeeping convenience. Unlike the other entries in this
           file, qew21/Genshin-Subtitles is not a binary or library that
           Kaption distributes at runtime; it is the historical ancestor
           of the current codebase.

                   source files derived from the upstream repository
                   have been modified. The codebase has been
                   substantially rewritten since the fork; references
                   to specific file-level provenance are not maintained

                   tree carry upstream copyright headers.

           §4(c) — Preservation of attribution. The upstream repository
